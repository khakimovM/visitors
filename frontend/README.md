# Visitor Track — Restoran Tashrif Buyuruvchilar Boshqaruv Tizimi

## ℹ️ Loyiha haqida

**Visitor Track** — restoran filiallariga tashrif buyuruvchilar oqimini real vaqt rejimida kuzatish, hisobotlarni olish va filiallar kesimida tahlil qilish tizimi.

🔑 **Asosiy imkoniyatlar**:

- Real-time odamlar oqimi
- Bugungi/haftalik/oylik tashrif statistikasi
- Filial bo‘yicha taqqoslash
- Sana oralig‘ida hisobot ko‘rish
- Jadval va grafik ko‘rinishida statistikalar
- PDF/Excel eksport qilish
- Qurilmalarni (Devices) boshqarish
- Filial va profil sozlamalari

---

## 🚀 Loyihani ishga tushirish

Loyihani o‘z kompyuteringizda ishga tushirish uchun quyidagilarni bajaring:

```sh
# 1. Reponi klon qiling
git clone <YOUR_GIT_URL>

# 2. Loyihaning papkasiga kiring
cd visitor-track

# 3. Kerakli kutubxonalarni o‘rnating
npm i

# 4. Development serverni ishga tushiring
npm run dev
```

---

## ✏️ Kodni tahrirlash usullari

**1. O‘z IDE’ingiz orqali**
Repo’ni klon qilib, VSCode yoki boshqa IDE’da tahrir qiling. Har qanday `git push` avtomatik ravishda yangilanishlarni aks ettiradi.

**2. GitHub’da to‘g‘ridan-to‘g‘ri**
Faylni tanlab, pencil tugmasi orqali tahrirlash mumkin.

**3. GitHub Codespaces**
Bir zumda cloud IDE orqali loyihani ishga tushirish mumkin.

---

## 🌍 Deploy qilish

Loyihani deploy qilish uchun quyidagi amallarni bajaring:

1. `npm run build` orqali production build yarating.
2. Hosil bo‘lgan `dist/` papkasini o‘zingiz tanlagan hosting (Vercel, Netlify, Render va h.k.) ga joylang.

---

## 🌐 Custom domain ulash

Agar loyiha deploy qilingan bo‘lsa, hosting provayderingiz (masalan, Vercel yoki Netlify) orqali o‘z domeningizni ulashingiz mumkin.
